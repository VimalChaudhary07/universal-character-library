üíª Code Examples

Creating an interactive character:
const character = await characterUtils.createInteractiveCharacter(
  'casual-boy',
  '#container',
  {
    onClick: (char) => char.play('wave'),
    onHover: (char, state) => {
      char.setSize(state === 'enter' ? 1.1 : 1);
    }
  }
);
Creating a scene with multiple characters:
const scene = await characterUtils.createScene('my-scene', [
  { type: 'casual-boy', container: '#scene' },
  { type: 'sporty-girl', container: '#scene' },
  { type: 'formal-man', container: '#scene' }
], {
  layout: 'circle',
  autoPlay: true
});
